# ISKCON Rajampet Website

This is a static website starter for ISKCON Rajampet.

## Folders
- `assets/images` for temple photos, deity photos, posters, and logo updates
- `assets/videos` for temple clips and festival videos
- `assets/css/styles.css` for design styling
- `assets/js/main.js` for interactions and festival cards
