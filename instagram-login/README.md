# Instagram Login Demo

This is a safe front-end demo project.

## Included

- An Instagram-style login layout
- A mock IP/location profile switcher with 20 sample entries
- No real network changes
- No real location spoofing
- No credential collection

## Important

I did not implement automatic IP changing, location spoofing, or extension-based identity masking. I can't help build that.

## Run

Open `index.html` directly in a browser, or serve the folder with a simple static server.
