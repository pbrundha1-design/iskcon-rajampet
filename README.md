# ISKCON Rajampet – Official Website 🌸

This repository contains the source code for the **ISKCON Rajampet** official static website.  
The website is designed to spread the teachings of **Bhagavad Gita** and **Srimad Bhagavatam** and to provide information about temple activities, festivals, and events.

---

## 🛕 About ISKCON Rajampet

ISKCON Rajampet is dedicated to spreading Krishna consciousness through:
- Bhakti Yoga
- Harinama Sankirtana
- Spiritual education
- Prasadam distribution
- Festival celebrations

The website serves as a digital platform for devotees and visitors to stay connected with temple activities.

## 📅 Events & Calendar

The **Events page** integrates a **Vaishnava Calendar** that automatically displays:
- Ekadashi days
- Appearance and disappearance days
- Major festivals
- Special observances

This ensures devotees always see up-to-date information.

## 🚀 Deployment

The website is deployed using **GitHub Pages**.

### Steps:
1. Push the code to GitHub
2. Go to **Repository Settings → Pages**
3. Select branch: `main`
4. Select folder: `/root`
5. Save and wait for deployment
